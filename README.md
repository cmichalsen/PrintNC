# PrintNC

## Compiling GRBLHal Firmware

Follow the GRBLHal platformio instructions:

- https://github.com/grblHAL/core/wiki/Compiling-grblHAL#platformio
- Use the webbuilder dev link so it generates the `platformio.ini` file.

## Prerequisites

1. Install PlatformIO Core:
   - https://docs.platformio.org/en/latest/core/installation/integration.html#step-2-install-platformio-core
2. Install the PlatformIO VS Code extension.
3. Clone the GRBLHal STM32F4xx repository:
   - `https://github.com/grblHAL/STM32F4xx`
4. Replace the `platformio.ini` file with the one generated by the webbuilder output.
5. Clone the referenced plugins and update the paths in `platformio.ini`.

Example `lib_deps` block:

```ini
lib_deps = ${common.lib_deps}
  eeprom
  ${wiznet_networking.lib_deps}
  C:/git/grblhal_probe_plugin
  C:/git/flexihal_atc
```

6. From the PlatformIO extension, run **Build**.
